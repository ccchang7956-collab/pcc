💡 GitHub Pages 備考門戶使用說明

  • 離線保存與可攜帶性：所有網頁均採用「離線自包含」架構設計。若要無網路環境下備考，只需在瀏覽器中按下 Ctrl+S (或 Mac Cmd+S) 另存為單一 HTML 網頁，即可帶到任何地方離線作答。
  • 行動裝置自適應 (RWD)：是非題文字陷阱已針對手機版面最佳化為「卡片流」，測驗按鈕設有大點擊熱區，方便您在通勤或零碎時間以手機流暢刷題。
  • 模擬測驗隨機挑戰：每次進入網頁或點擊「隨機重測」時，測驗系統都會將所有題目與選項重新隨機打亂，確保您能真正理解法規邏輯，而非死背題序答案。
