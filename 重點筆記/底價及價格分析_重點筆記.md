## 🎯 第一板塊：核心知識架構

### 1. 底價訂定之權責歸屬與程序
*   **核定權限**：一般由規劃、設計、需求或使用單位提出預估金額及其分析後，由承辦採購單位簽報**機關首長或其授權人員**核定。
*   **例外簡化程序**：**重複性採購**或**未達公告金額之採購**，得由**承辦採購單位逕行簽報核定**（得免由需求單位預估）。
*   **異動與效力**：因機關首長異動，原已核定之底價**不需重新訂定**，原核定底價依然有效。
*   **監辦與實務**：開標主持人開啟底價封後，監辦人員**查看底價金額尚無不可**，但仍需遵守決標前之保密規定。

### 2. 底價訂定之法定時限
*   **公開招標（分段開標）**：至遲應於**價格標開標前**定之。
*   **選擇性招標**：應於資格審查後之**下一階段（價格標/比價）開標前**定之。
*   **限制性招標（比價）**：應於**辦理比價之開標前**定之。
*   **限制性招標（議價）**：訂定底價前，**應先參考廠商之報價或估價單**（包含公開客觀評選優勝廠商後之議價）。
*   **當場改採議價（未達公告金額）**：當場改採議價時，其底價訂定前**仍應先參考廠商之報價或估價單**。
*   **公開取得書面報價**：底價應於**進行比價或議價前**定之（非開標前）。

### 3. 底價之公開與保密原則
*   **保密時機**：底價於開標前與決標前應予**保密**；決標後應予**公開**。
*   **不公開之例外（商業機密）**：若屬**以轉售或供製造成品以供轉售之採購**，其底價涉及商業機密者，決標後**得不予公開**，但應通知**得標廠商**。
*   **機密採購例外**：軍事機密採購之底價於決標後得不予公開。
*   **複數決標之公開時機**：採用複數決標方式，尚有部分項目未決標時，其**已決標部分之底價應予公開**。
*   **流標與廢標之底價處理**：流標後因底價未開啟，若招標條件均未改變，底價**經機關首長或其授權人核准免再重新訂定（得沿用）**；但若廢標後重行招標，且圖說、規範、成本、行情有改變者，應重行訂定。

### 4. 價格與成本分析實務
*   **成本分類**：
    *   **固定成本**：指**不隨**履約標的之數量而變動之成本（如折舊、基本設立費）。
    *   **變動成本**：會隨履約標的數量而變動之成本（如原材料、直接人工）。
*   **利潤合理編列**：利潤得視個案，依所需成本、市場之競爭性及經營策略加以分析，合理編列，避免超額利潤。
*   **不訂底價之替代審查**：準用最有利標評選優勝廠商，議價階段如不訂底價，可成立**評審委員會**審查廠商報價；當認為標價合理者，得不提出建議減價金額，直接照得標廠商之報價決標。

## 🔢 第二板塊：常考「數字、比例與時限」密碼

| 關鍵數字/比例/時限 | 適用法規與實務情境說明 |
| :--- | :--- |
| **4%** | 超底價決標限制（最低標標價超過底價 **4%** 以內，且未逾預算金額，查核金額以上採購需先報經上級機關核准後決標）。 |
| **8%** | 超底價決標上限（緊急情事需決標時，未達公告金額或重複性採購，經機關首長或授權人員核准可以在超底價 **8%** 以內決標；不論如何決標金額**絕對不得超過預算金額**）。 |
| **80%** | 標價偏低認定門檻（訂有底價之採購，廠商總標價低於底價 **80%**，或部分標價低於該部分項目底價之 **80%** 者，得認定為標價偏低）。 |
| **70%** | 無底價與其他機關參考值（無底價且未設評審會之採購，總標價低於預估或預估需用金額 **70%** 屬偏低；部分標價低於其他機關最近相同採購決標價或一般價格 **70%** 亦同）。 |
| **1,000 萬元** | 價格資料庫傳輸門檻（預算金額達新台幣 **1,000 萬元** 以上之工程採購，應於決標後一定期限內將契約單價及預算單價資料傳輸至工程價格資料庫）。 |
| **3 次** | 重新比減價格次數上限（最低標超底價優先減價 **1 次**，減價結果仍超底價時，比減價格不得逾 **3 次**）。 |
| **1 辦公日** | 幣別折算基準日（標價幣別在2種以上者，依**辦理決標前 1 辦公日**臺灣銀行收盤即期賣出匯率折算）。 |

## 📝 第四板塊：精選經典題解析

### 1. 優先減價與比減價之參與權利 (選擇題 34)
*   **題目**：最低標未進入底價，當場表示放棄優先減價，該廠商得否參與後續比減價？
*   **答案**：**得**。
*   **核心解析**：放棄「優先減價」僅代表廠商放棄單獨減價之優惠權利，但在後續所有廠商共同參與的「比減價」程序中，該最低標廠商依然合於招標文件規定，具有完全平等的減價權利，機關不可剝奪。

### 2. 多種幣別折算匯率基準日 (是非題 94)
*   **題目**：投標廠商之標價幣別在2種以上，由新臺幣折算總價以定標序。該折算總價，依辦理開標前1辦公日臺灣銀行即期賣出匯率折算之。
*   **答案**：**X (錯誤)**。
*   **核心解析**：依政府採購法施行細則，折算總價應以**辦理決標前 1 辦公日**臺灣銀行外匯交易收盤即期賣出匯率折算，而非開標前 1 辦公日。這是極細微的階段字眼陷阱。

### 3. 採購法第58條之差額保證金實務判定 (選擇題 44)
*   **題目**：最低標廠商標價410萬，低於底價600萬之80%（480萬）與70%（420萬）。機關認為廠商提出之說明「尚非完全合理」，關於差額保證金之通知，何者正確？
*   **答案**：**不通知該廠商提出**。
*   **核心解析**：依政府採購法第58條執行程序，若廠商之說明「合理」，則照價決標；若說明「尚非完全合理」，且機關判定其降低品質、不能誠信履約之虞，則**不通知廠商繳納差額保證金，直接不予決標予該廠商**。只有在說明「尚非完全合理，但廠商願意繳納差額保證金」且機關同意時，才會通知其繳納。

### 4. 公告底價之核准程序 (是非題 59)
*   **題目**：機關得依實際需要，報經上級機關核准後，於招標文件公告底價。
*   **答案**：**X (錯誤)**。
*   **核心解析**：公告底價是招標機關的裁量權限，機關得依實際需要，**自行決定**於招標文件中公告底價，**完全不需報經上級機關核准**。

### 5. 最有利標與價格協商洽減 (選擇題 4 & 41)
*   **題目**：以最有利標決標且不訂底價者，招標文件無需載明評選項目？
*   **答案**：**X (錯誤)**。
*   **核心解析**：不論是否訂有底價，最有利標之評選皆必須在招標文件中明確載明評選項目。若最有利標定有底價，且廠商報價超底價，則於**協商措施時洽減之**（不得在議價或評選時洽減）。
